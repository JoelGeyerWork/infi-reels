# אינפי · רילס — לימוד משפטים והוכחות בסגנון רילס

אפליקציית web סטטית ללימוד משפטים והוכחות באינפי. כל "ריל" הוא משפט; הקשה הופכת את הכרטיס וחושפת את ההוכחה. גלילה אנכית (snap) למשפט הבא, בעברית RTL, עם רינדור נוסחאות ב-KaTeX.

## תכונות
- **הפיכת כרטיס**: צד אחד המשפט, צד שני ההוכחה.
- **גלילת רילס** אנכית עם הצמדה (scroll-snap).
- **סינון לפי פרק** ומונה התקדמות.
- **סימון כנלמד** (נשמר ב-`localStorage`).
- **הערות תיקון** (⚠) במקומות שבהם תוקנה טעות לכאורה במקור.

## הרצה מקומית
כל שרת סטטי מספיק (צריך שרת בגלל `fetch` של `theorems.json`):

```bash
python3 -m http.server 8765
# ואז לפתוח http://127.0.0.1:8765
```

## פריסה ל-Vercel
אתר סטטי ללא שלב build. מהתיקייה:

```bash
npx vercel        # תצוגה מקדימה
npx vercel --prod # פרודקשן
```

לחלופין: לדחוף ל-GitHub ולייבא את הריפו ב-vercel.com (Framework Preset: **Other**).

## מבנה
- `index.html` · `app.js` · `styles.css` — האפליקציה.
- `theorems.json` — התוכן: `{ ref, chapter, statement, proof, title?, note? }`, עם מתמטיקה ב-LaTeX בתוך `$…$` / `$$…$$`.
